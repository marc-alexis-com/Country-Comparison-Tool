Documentation

Overview

This application allows users to input a target population and/or area to find the top 3 countries closest to those values. It's designed with a retro Apple System OS interface using System.css.

How to Use

  1. Navigate to the Home Page.
  2. Enter the desired population and select the appropriate unit (Units, Thousands, Millions, Billions).
  3. Enter the desired area (km2) and select the appropriate unit.
  4. Click the Compare button to view the results.

Features

  • Responsive design that adapts to different screen sizes.
  • Retro Apple System OS look using System.css.
  • Ability to find countries closest in population and area to user-defined values.
  • Easy unit selection via dropdown menus.

Technical Details

The application is built using Python and Flask for the backend, and utilizes the Rest Countries API to fetch country data. It caches data locally to improve performance and reduce API calls.

Dependencies

  • Python 3.6 or higher
  • Flask
  • Requests
  • System.css
