About

About This Application

This Country Comparison Tool was developed to provide users with an easy way to find countries that are closest in population and area to specified values. The application combines data retrieval with a nostalgic user interface inspired by the classic Apple System OS.

Developer Information

Created by Marc-Alexis, a passionate developer interested in data visualization and retro UI design.

Technologies Used

  • Flask: A lightweight WSGI web application framework for Python.
  • System.css: A CSS library for building retro-inspired interfaces resembling Apple's System OS.
  • Rest Countries API: Provides detailed information about countries in a JSON format.
